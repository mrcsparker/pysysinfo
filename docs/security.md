# Security

The project currently supports security fixes for the latest released minor
version of `pysysinfo`.

The canonical security policy lives at
[`SECURITY.md`](https://github.com/mrcsparker/pysysinfo/blob/main/SECURITY.md).

## Supported Versions

| Version | Supported |
| --- | --- |
| `0.2.x` | Yes |
| `< 0.2.0` | No |

## Reporting

Please report vulnerabilities through a private GitHub security advisory:

- [Open a private advisory](https://github.com/mrcsparker/pysysinfo/security/advisories/new)

For non-sensitive issues that do not disclose exploit details, a normal GitHub
issue is fine.
