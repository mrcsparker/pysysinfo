# Security Policy

## Supported Versions

The latest released minor version receives security fixes.

| Version | Supported |
| --- | --- |
| `0.2.x` | Yes |
| `< 0.2.0` | No |

## Reporting a Vulnerability

Please report suspected vulnerabilities by opening a private security advisory
on GitHub:

- [github.com/mrcsparker/pysysinfo/security/advisories/new](https://github.com/mrcsparker/pysysinfo/security/advisories/new)

If private advisory tooling is unavailable, open an issue only for
non-sensitive problems that do not disclose exploit details.

When reporting a vulnerability, include:

- the affected version
- the operating system and Python version
- reproduction steps or a minimal proof of concept
- any known impact or scope

We will acknowledge a valid report as quickly as practical, work on a fix, and
coordinate disclosure once the issue is understood and patched.
